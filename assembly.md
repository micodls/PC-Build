# Assembly

## Tools
---
* Screwdriver/s (preferrably magnetic)
* Long nose pliers

## Motherboard
---
1. Prepare all your parts.
   <img src="./assets/hardware/Group Photo.jpg" alt="drawing" width="500"/>
2. Unbox the ASUS Strix z490-i motherboard. Inspect for damages and learn all the connector ports.
   <img src="./assets/hardware/Motherboard/Motherboard.jpg" alt="drawing" width="500"/>
3. Install the Intel i5-10600k and the Corsair Vengeance RAM sticks.
   <img src="./assets/hardware/Motherboard/Motherboard (No CPU).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/Motherboard (with CPU).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/Motherboard (with CPU and RAM).jpg" alt="drawing" width="500"/>
4. Hard part! Replacing the stock Intel WiFi card with the BCM943602CS. First, Remove the triple deck heatsink for the NVME slots. Next, remove the following screws from the back of the motherboard.
   <img src="./assets/hardware/Motherboard/BCM : NVME/Motherboard (Bottom).jpg" alt="drawing" width="500"/>
5. Before completely removing the Heatsink / I/O shield, carefully remove the VRM fan header and the tape.
   <img src="./assets/hardware/Motherboard/BCM : NVME/VRM Fan (Not removed).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/BCM : NVME/VRM Fan (Removed).jpg" alt="drawing" width="500"/>
6. Take note of the placement of the VRM Heatsink thermal pads!
   <img src="./assets/hardware/Motherboard/BCM : NVME/Heatsink w: Thermal Pads (Bottom view).jpg" alt="drawing" width="500"/>
7. Set aside the Heatsink / I/O shield. Remove the additional screw for the Intel WiFi chip.
   <img src="./assets/hardware/Motherboard/BCM : NVME/WiFi Chip additional screw.jpg" alt="drawing" width="500"/>
8. Carefully pull out the Intel WiFi chip.
   <img src="./assets/hardware/Motherboard/BCM : NVME/WiFi Card (Removed).jpg" alt="drawing" width="500"/>
9. Open the Intel WiFi chip and carefully pry off the antenna cables from the chip. Replace the antenna cables with the ones included in the WiFi Antenna Set.
   <img src="./assets/hardware/Motherboard/BCM : NVME/WiFi Chip (Opened).jpg" alt="drawing" width="500"/>
10. Before completely closing the Intel WiFi chip, bend the top a little bit so that the longer cables have a place to pass thru.
   <img src="./assets/hardware/Motherboard/BCM : NVME/WiFI cables (Replaced).jpg" alt="drawing" width="500"/>
11. Return everything starting from the Intel WiFi chip, the additional screw, VRM fan header, heatsink and the bottom screws.
   <img src="./assets/hardware/Motherboard/BCM : NVME/Heatsink (Returned).jpg" alt="drawing" width="500"/>
12. Install the Samsung 970 evo NVME ssd on the top side.
   <img src="./assets/hardware/Motherboard/BCM : NVME/NVME (Top).jpg" alt="drawing" width="500"/>
13. Assemble the BCM943602CS and B+M Adapter. Install this on the bottom side.
   <img src="./assets/hardware/Motherboard/BCM : NVME/BCM + Adapter (Bottom).jpg" alt="drawing" width="500"/>
14. Attach the bluetooth antenna cable and WiFi antenna cables.
   <img src="./assets/hardware/Motherboard/BCM : NVME/BCM (w: Bluetooth Antenna).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/BCM : NVME/BCM with Wifi Cables.jpg" alt="drawing" width="500"/>
15. Attach the USB 2 header on the B+M Adapter and motherboard.
   <img src="./assets/hardware/Motherboard/BCM : NVME/BCM Chip w: USB 2 Header.jpg" alt="drawing" width="500"/>
16. The bottom heatsink should be set aside since it won't fit due to the BCM943602CS and B+M Adapter. Reassemble the triple deck heatsink for the NVME slots. Make sure you remove the thermal pad cover for the top part before installing everything back. Tip: Tuck the cables inside the heatsink for better cable management. Just be sure to not bend the cables to much. Hard part is done!
   <img src="./assets/hardware/Motherboard/BCM : NVME/NVME Thermal Pad.jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/BCM : NVME/NVME : BCM (Top view).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/BCM : NVME/NVME : BCM (Side view).jpg" alt="drawing" width="500"/>
17. Install the CPU cooler mounts.
   <img src="./assets/hardware/Motherboard/CPU Cooler Rear Mount.jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Motherboard/CPU Cooler Front Mount.jpg" alt="drawing" width="500"/>

## Case
---
1. Install the power supply.
   <img src="./assets/hardware/Case/Power Supply/Power Supply (Front).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Case/Power Supply/Power Supply (Rear).jpg" alt="drawing" width="500"/>
2. Install the bottom case fan/s (intake).
   <img src="./assets/hardware/Case/Bottom Case Fan.jpg" alt="drawing" width="500"/>
3. Install the rear case fan (exhaust).
   <img src="./assets/hardware/Case/Rear Case Fan.jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/Case/Case.jpg" alt="drawing" width="500"/>
4. Install the motherboard in the case.
   <img src="./assets/hardware/Case/Motherboard In Case.jpg" alt="drawing" width="500"/>
5. Attach the front I/O cables, and cable extensions (if you have).
   <img src="./assets/hardware/Case/Motherboard (w: Front Cables and Cable Extensions).jpg" alt="drawing" width="500"/>
6. Install the GPU.
   <img src="./assets/hardware/Case/GPU w:o Cables.jpg" alt="drawing" width="500"/>
7. Assemble the SATA SSD/s.
   <img src="./assets/hardware/Case/Sata SSD.jpg" alt="drawing" width="500"/>
8. Before installing the SATA SSD/s in the case, make sure that the SATA power and data cables are already attached.
   <img src="./assets/hardware/Case/SSD w: Cable.jpg" alt="drawing" width="500"/>
9. Mount the SSD/s in your case.
   <img src="./assets/hardware/Case/SSD Mounted.jpg" alt="drawing" width="500"/>
10. Attach all the PSU cables.
   <img src="./assets/hardware/Case/GPU w: Cables.jpg" alt="drawing" width="500"/>
11. Install the front case fan (intake).
   <img src="./assets/hardware/Case/Amost done w:o  CPU cooler.jpg" alt="drawing" width="500"/>
12. Apply a pea sized thermal paste on your CPU.
   <img src="./assets/hardware/Case/Pea size Thermal Paste.jpg" alt="drawing" width="500"/>
13. Install the CPU Cooler heatsink without the fans.
   <img src="./assets/hardware/Case/CPU Cooler w:o Fans.jpg" alt="drawing" width="500"/>
14. Install the CPU Cooler fans. And you're done!
   <img src="./assets/hardware/Case/With CPU Cooler Fans and Front Intake.jpg" alt="drawing" width="500"/>
15. Boot and check in your BIOS if everything is detected.
   <img src="./assets/hardware/BIOS (Advanced).jpg" alt="drawing" width="500"/>
   <img src="./assets/hardware/BIOS (Easy).jpg" alt="drawing" width="500"/>
